# 3D Models Folder

Place your downloaded Sketchfab models here!

## Recommended File Names:
- `castle.glb` - Fantasy castle
- `tree.glb` - Low poly tree
- `fantasy_house.glb` - Fantasy house
- `pirate_ship.glb` - Pirate ship
- `tower.glb` - Watch tower
- `dragon.glb` - Dragon
- `windmill.glb` - Windmill

## Where to Get Free Models:

### 1. Sketchfab (requires download)
- https://sketchfab.com/
- Filter: "Downloadable" + "CC License"
- Format: GLB or GLTF

### 2. Poly Pizza (CC0 - Free)
- https://poly.pizza/
- All models are free, no attribution required
- Direct GLB downloads

### 3. Kenney Assets (CC0 - Free)
- https://kenney.nl/assets
- Pre-made game assets
- Excellent for low-poly style

### 4. Quaternius (CC0 - Free)
- https://quaternius.com/packs.html
- Ultimate Low Poly Pack
- Perfect for voxel worlds

## Quick Test
Once you download a model, rename it to match the names above and refresh your browser!

Example:
1. Download a castle from Sketchfab
2. Save as `castle.glb` in this folder
3. Refresh browser
4. Open console (F12)
5. Type: `window.appInstance.island.loadSketchfabModel('castle')`
6. Watch your castle appear! üè∞
